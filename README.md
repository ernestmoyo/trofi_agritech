# Trofi AgriTech — Agriculture Redefined

A women-led agricultural consulting firm delivering comprehensive data analytics, training, and advisory services across Africa's agricultural sector — serving policymakers, researchers, development agencies, and agribusiness stakeholders across all 55 AU member states.

**Founded by Isabel Moyo | Dar es Salaam, Tanzania**

## What We Do

Trofi AgriTech bridges Africa's agricultural data silos, empowers women in agriculture, and delivers world-class consulting and training services that drive evidence-based decision-making across the continent's food systems.

## Our Six Pillars

1. **Agriculture Policy Tracker** — Track, compare, and analyze agriculture policies across all 55 AU member states
2. **Agriculture Trade Statistics** — Track intra-African and global agriculture trade flows
3. **Post-Harvest Loss Tracker** — Monitor and analyze post-harvest losses across value chains and geographies
4. **Value Chain Analytics** — Map and analyze agriculture value chains across the continent
5. **AI Adoption Tracker** — Track AI and digital technology adoption and impact in African agriculture
6. **Training & Capacity Building** — Empowering women and youth through agricultural training, consulting, and leadership development programs

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Icons:** Lucide React
- **Deployment:** GitHub Pages with custom domain

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the platform locally.

## Build & Deploy

```bash
npm run build
```

Static files are generated in the `out/` directory. Deployment to [trofi-agritech.org](https://trofi-agritech.org) is automated via GitHub Actions on every push to `main`.
